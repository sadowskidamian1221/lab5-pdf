# Lab 5 – Biblioteki PDF w Pythonie

Dziedzina: obsługa plików PDF  
Biblioteki:
- ReportLab – tworzenie PDF
- PyPDF2 – łączenie PDF i odczyt tekstu

## Pliki
- raport.md – opis bibliotek 
- examples create_pdf.py – tworzy a_reportlab.pdf i b_reportlab.pdf do folderu output
- examples merge_pdf.py – łączy PDF-y w output merged.pdf

## Instalacja
python -m pip install reportlab PyPDF2

## Uruchomienie
python examples create_pdf.py
python examples merge_pdf.py
